# כרטיס רכב (שם עבודה) — חבילת תכנון

שדרוג הפרויקט `IL-Car-Details-Finder`: מסקריפט CLI → **אתר יפה, רציני ונגיש** לבדיקת רכב לפי מספר רישוי.

> **כיוון (מעודכן):** פרויקט אישי, **לא מוצר מסחרי**. חינמי לחלוטין — אין לידים, דוחות בתשלום, פרסום, מנויים,
> חשבונות או מסד נתונים. הכל צד‑לקוח. דגש על קונספט עבודה חכם, עיצוב מוקפד, ו**נגישות WCAG 2.1 AA מאומתת**.

## הכרעות מפתח
- **אפס עלות:** Vite+React, `localStorage` לשמירה, קובץ `.ics` לתזכורת יומן, אירוח חינמי. אין שרת/DB/מייל.
- **נגישות = חובה ומאומתת:** axe נקי, Lighthouse a11y ≥95, ניווט מקלדת, קורא‑מסך — שער חובה ב‑M6.
- **נתונים:** data.gov.il (רשמי, חינמי); מציגים בשקיפות מה זמין ומה לא.

## בדיקת נגישות שכבר בוצעה
כל זוגות הצבע במוקאפ נבדקו מול WCAG AA: טקסט ראשי 16.8:1, משני 7.6:1, כפתור לבן על כחול 5.2:1, סטטוסים
5.4–5.9:1, גבול שדה קלט 3.1:1 — **הכל עובר**. הסטטוסים משודרים בטקסט+אייקון, לא רק בצבע.
